## 🛠️ 시작하기 전 준비물
### 1. GitHub 계정 만들기
- [GitHub.com](https://github.com) 가서 회원가입
- 무료로 만들면 돼요!

### 2. VS Code 설치하기
- [VS Code 다운로드](https://code.visualstudio.com/)
- "Download for Windows" 클릭해서 설치

### 3. Git 기본 개념 알기
- **Commit**: 내가 작성한 코드를 저장하는 것
- **Push**: 내 컴퓨터에서 GitHub로 업로드하는 것
- **Branch**: 나만의 작업 공간 만들기

## 🚀 스터디 참여하는 방법

### 📥 1단계: 처음 시작하기
```bash
# 1. 프로젝트 다운로드 받기
git clone https://github.com/reomoon/study_python.git
cd study_python

# 2. 문제 받기 (1주차 예시)
git checkout template/week1

# 3. 내 작업 공간 만들기 (본인 이름으로 바꾸기!)
git checkout -b study/본인이름
# 예시: git checkout -b study/철수
```

### ✏️ 2단계: 문제 풀기
1. **VS Code로 파일 열기**
   - `week1_variable.py` 파일을 더블클릭
   
2. **답안 작성하기**
   ```python
   # 이런 식으로 빈 곳에 코드 작성하기
   # 답을 아래에 작성하세요.
   x = 7  # 여기에 답안 작성!
   print("x =", x)
   ```

3. **내 답안 확인하기**
   ```bash
   # 터미널에서 실행해보기
   python week1_variable.py
   
   # 자동 채점하기
   python test_checker.py
   ```

### 📤 3단계: 제출하기
```bash
# 1. 내가 수정한 파일들 추가하기
git add .

# 2. 설명과 함께 저장하기
git commit -m "1주차 변수 문제 완료"

# 3. GitHub에 업로드하기
git push origin study/본인이름
```

### 🔄 4단계: 새 주차 시작하기
```bash
# 새로운 주차 문제 받기 (2주차 예시)
git checkout template/week2
git checkout study/본인이름
git merge template/week2
```
