1주차: 변수 문제

  1. 1) 변수 x에 7을 저장하세요.
    2) print()로 x의 값을 출력하세요.
    예: x = 7; print(x) → 7 출력
  2. 1) 변수 a와 b에 각각 3과 8을 저장하세요.
    2) 두 변수의 곱(a * b)을 print()로 출력하세요.
    예: a = 3; b = 8; print(a * b) → 24 출력
  3. 1) 변수 name에 자신의 이름을 저장하세요.
    2) print()로 "내 이름은 OOO입니다."를 출력하세요.
    예: name = '철수'; print(f"내 이름은 {name}입니다.")
  4. 1) 변수 width와 height에 각각 5와 10을 저장하세요.
    2) 사각형의 넓이(width * height)를 print()로 출력하세요.
    예: print(width * height) → 50 출력
  5. 1) 변수 temp에 36.5를 저장하세요.
    2) print()로 "현재 체온은 36.5도입니다."와 같이 출력하세요.
    예: print(f"현재 체온은 {temp}도입니다.")
힌트:
  • 변수는 = 기호로 값을 저장합니다.
  • print() 함수로 값을 출력할 수 있습니다.
  • 문자열은 큰따옴표(") 또는 작은따옴표(')로 감쌉니다.
  • 곱셈은 * 연산자를 사용합니다.
  • 여러 값을 출력할 때는 ,로 구분할 수 있습니다.
