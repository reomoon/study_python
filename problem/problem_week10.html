10주차 - 클래스

  1. 간단한 클래스 Person를 정의하고 이름(name)과 나이(age)를 속성으로 가지게 하세요.
  2. greet 메서드를 만들어 "안녕하세요, 제 이름은 OOO입니다"를 반환하도록 하세요.
  3. 클래스 인스턴스를 생성하고 메서드를 호출한 결과를 출력하세요.
  4. 상속을 사용해 Student 클래스를 만들고 추가 속성(예: school)을 넣어보세요.
  5. __str__ 또는 __repr__을 구현하여 문자열 표현을 제공하세요.
  6. 여러 학생(Student) 인스턴스를 리스트로 관리하는 Classroom 클래스를 만들고, 학생 추가/전체 학생 출력 메서드를 구현하세요.
    (클래스 변수/메서드 활용, 반복문/객체 리스트 다루기)
힌트:
  • class로 클래스를 정의합니다.
  • __init__은 생성자, self는 인스턴스 자신을 가리킵니다.
  • def로 메서드(함수)를 만듭니다.
  • super()로 부모 클래스의 기능을 사용할 수 있습니다.
  • __str__, __repr__로 객체 출력 형식을 지정할 수 있습니다.
  • 여러 객체를 리스트에 담아 반복문으로 처리할 수 있습니다.
  • 클래스 내부에서 리스트를 속성으로 관리하면 여러 인스턴스를 저장할 수 있습니다.
  • 학생 추가는 append()로, 전체 학생 출력은 반복문과 __str__ 활용.
