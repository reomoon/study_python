10주차 - 클래스

  1. 1) Person 클래스를 정의하고, 이름(name)과 나이(age)를 속성으로 저장하세요.
    예: Person("철수", 17)
  2. 2) greet라는 메서드를 만들어 "안녕하세요, 제 이름은 OOO입니다"를 반환하도록 하세요.
    예: p.greet() → "안녕하세요, 제 이름은 철수입니다"
  3. 3) Person 인스턴스를 생성하고, 메서드를 호출한 결과를 출력하세요.
    예: print(p.greet())
  4. 4) Student 클래스를 Person을 상속해서 만들고, 학교(school) 속성을 추가하세요.
    예: Student("영희", 16, "서울여고")
  5. 5) __str__ 또는 __repr__을 구현해서 객체를 print할 때 "이름(나이) - 학교"처럼 출력되게 하세요.
    예: print(s) → "영희(16) - 서울여고"
  6. 6) 여러 학생(Student) 인스턴스를 리스트로 관리하는 Classroom 클래스를 만들고, 학생 추가/전체 학생 출력 메서드를 구현하세요.
    (클래스 변수/메서드 활용, 반복문/객체 리스트 다루기)
    예: c.add_student(Student(...)), c.show_students()
힌트:
  • class로 클래스를 정의합니다.
  • __init__은 생성자, self는 인스턴스 자신을 가리킵니다.
  • def로 메서드(함수)를 만듭니다.
  • super()로 부모 클래스의 기능을 사용할 수 있습니다.
  • __str__, __repr__로 객체 출력 형식을 지정할 수 있습니다.
  • 여러 객체를 리스트에 담아 반복문으로 처리할 수 있습니다.
  • 클래스 내부에서 리스트를 속성으로 관리하면 여러 인스턴스를 저장할 수 있습니다.
  • 학생 추가는 append()로, 전체 학생 출력은 반복문과 __str__ 활용.
