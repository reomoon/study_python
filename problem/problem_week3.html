3주차 - 출력/입력

  1. 화면에 "Hello, world!"를 출력하세요.
  2. 사용자에게 이름을 입력받고, "안녕하세요, OOO님" 형식으로 출력하세요. input()을 사용하세요.
  3. 두 숫자를 입력받아 더한 결과를 출력하세요. input()과 int()를 활용하세요.
  4. 파일에서 읽지 않고, print()문을 이용하여'FASHIONGO'문구를 여러 줄로 출력하세요. 여러 번 호출하거나 반복문, \n을 활용하세요.
  5. 사용자 입력을 int()로 변환하여 계산한 후 결과를 출력하세요.
힌트:
  • input()으로 입력받은 값은 항상 문자열(str)입니다.
  • 숫자 계산을 하려면 int()로 변환하세요.
  • 여러 줄을 출력하려면 print()를 여러 번 사용하거나 \n을 활용할 수 있습니다.
  • f-string: print(f"안녕하세요, {name}님")처럼 쓸 수 있습니다.
