5주차 - 반복문

  1. for 문을 사용해 1부터 5까지 출력하세요.
  2. 리스트의 각 요소를 반복해 합을 계산하고 출력하세요.
  3. while문을 사용해 간단한 카운트다운을 출력하세요.
  4. 리스트 컴프리헨션(한 줄로 리스트 만들기)을 사용해서, 1부터 5까지 각 숫자의 제곱을 담은 리스트를 만들어 출력하세요.
    예시: [x*x for x in range(1, 6)] 결과는 [1, 4, 9, 16, 25] 입니다.
  5. for문을 사용해 2단부터 9단까지 구구단을 출력하세요.
  6. 반복문에서 break나 continue를 사용한 예를 작성하세요.
힌트:
  • for, while로 반복문을 만들 수 있습니다.
  • range()로 숫자 반복, enumerate()로 인덱스와 값 동시 사용.
  • break, continue로 반복 제어.
  • 리스트 컴프리헨션은 [x*x for x in list]처럼 사용합니다.
  • sum()으로 리스트의 합을 구할 수 있습니다.
  • 구구단은 중첩 for문으로 만들 수 있습니다.
  • f"{i} x {j} = {i*j}"처럼 f-string을 사용하면 변수 값을 문자열에 쉽게 넣을 수 있습니다.
