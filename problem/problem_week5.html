5주차 - 반복문

  1. 1) for문으로 1부터 5까지 숫자를 한 줄씩 출력하세요.
    예: 1, 2, 3, 4, 5 출력
  2. 1) 리스트를 만들고, 각 요소를 반복문으로 더해서 합계를 출력하세요.
    예: [1,2,3,4,5] → 15 출력
  3. 1) while문을 사용해 3부터 0까지 카운트다운을 출력하세요.
    예: 3, 2, 1, 0 출력
  4. 1) 리스트 컴프리헨션으로 1~5의 제곱값을 담은 리스트를 만들어 출력하세요.
    2) 예시: [x*x for x in range(1, 6)] → [1, 4, 9, 16, 25] 출력
  5. 1) for문을 중첩해서 2단부터 9단까지 구구단을 출력하세요.
    2) 각 줄에 f"{i} x {j} = {i*j}" 형태로 출력하세요.
    예: 2 x 1 = 2 ... 9 x 9 = 81
  6. 1) 반복문에서 break 또는 continue를 사용한 예시 코드를 작성하세요.
    예: 2일 때 continue, 4일 때 break 등
힌트:
  • for, while로 반복문을 만들 수 있습니다.
  • range()로 숫자 반복, enumerate()로 인덱스와 값 동시 사용.
  • break, continue로 반복 제어.
  • 리스트 컴프리헨션은 [x*x for x in list]처럼 사용합니다.
  • sum()으로 리스트의 합을 구할 수 있습니다.
  • 구구단은 중첩 for문으로 만들 수 있습니다.
  • f"{i} x {j} = {i*j}"처럼 f-string을 사용하면 변수 값을 문자열에 쉽게 넣을 수 있습니다.
