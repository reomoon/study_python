2주차 - 데이터 타입과 기본 연산

  1. 1) 변수 num에 42를 저장하세요.
    2) type()으로 num의 타입을 출력하세요.
    예: num = 42; print(type(num)) → <class 'int'> 출력
  2. 1) 변수 a에 15, b에 4를 저장하세요.
    2) a + b의 결과를 print()로 출력하세요.
    예: print(a + b) → 19 출력
  3. 1) 변수 name에 자신의 이름을 문자열로 저장하세요.
    2) print()로 "내 이름은 ..." 형식으로 출력하세요.
    예: name = '영희'; print(f"내 이름은 {name}입니다.")
  4. 1) 리스트 numbers를 만들고, 숫자를 여러 개 넣으세요.
    2) len(numbers) 또는 sum(numbers)로 리스트의 길이나 합을 출력하세요.
    예: numbers = [1,2,3]; print(len(numbers)), print(sum(numbers))
  5. 1) 딕셔너리 info를 만들고, 나이(age)와 도시(city)를 키로 저장하세요.
    2) info['age']로 나이를 출력하세요.
    예: info = {'age': 20, 'city': '서울'}; print(info['age'])
힌트:
  • print() 함수로 값을 출력할 수 있습니다.
  • type() 함수로 변수의 타입을 확인할 수 있습니다.
  • 덧셈은 + 연산자를 사용합니다.
  • 문자열은 큰따옴표(") 또는 작은따옴표(')로 감쌉니다.
  • 리스트는 []로, 딕셔너리는 {}로 만듭니다.
  • 리스트의 길이는 len(), 합은 sum()으로 구할 수 있습니다.
