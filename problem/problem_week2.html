2주차 - 데이터 타입과 기본 연산

  1. 변수 num에 정수값 42를 할당하고, 타입을 출력하세요.
    (예: print(type(num))) 
    num = 42
print(type(num))  # <class 'int'> 가 출력됩니다
  2. 변수 a에 15, b에 4를 할당하고, 덧셈 결과를 출력하세요.
    (예: print(a + b)) 
    a = 15
b = 4
print(a + b)  # 19 출력
  3. 변수 name에 문자열로 자신의 이름을 저장하고 "내 이름은 ..." 형식으로 출력하세요. 
    name = '홍길동'
print(f"내 이름은 {name}입니다.")
  4. 리스트 numbers를 만들고 몇 가지 숫자를 넣은 뒤, 리스트의 길이나 합을 출력하세요. 
    numbers = [1, 2, 3, 4]
print(len(numbers))  # 길이 출력
print(sum(numbers))  # 합 출력
  5. 딕셔너리 info에 나이(age)와 도시(city)를 포함시켜 저장하고, 나이를 출력하세요. 
    info = {'age': 25, 'city': 'Seoul'}
print(info['age'])  # 25 출력
