9주차 - 파일 입출력

  1. 0) fn = 'examples_week9.txt'처럼 파일명을 변수로 선언하세요.
    실제 파일이 없어도 변수만 있으면 됩니다.
  2. 1) 텍스트 파일을 생성하고, write()로 여러 줄의 텍스트를 저장하는 코드를 작성하세요.
    예: 'line1\nline2\n' 저장
  3. 2) 저장한 파일을 open()으로 열고, read() 또는 readlines()로 내용을 읽어 출력하세요.
    예: 파일 내용 전체 또는 각 줄 출력
  4. 3) 파일이 없을 때 try/except로 예외를 처리하는 코드를 작성하세요.
    예: FileNotFoundError 발생 시 "파일이 없습니다." 출력
  5. 4) 파일의 각 줄을 readlines()로 리스트에 저장하고, 리스트의 길이를 출력하세요.
    예: ['line1', 'line2'] → 2 출력
  6. 5) 파일을 'a' 모드로 열어 한 줄을 덧붙이는 코드를 작성하세요.
    예: 'line3' 추가
  7. 6) 예제 실행이 끝난 뒤 os로 생성한 파일을 삭제하는 코드를 작성하세요.
    폴더를 깨끗하게 유지하기 위해 사용합니다. 이때 os 모듈은 첫 줄에 import되어 있어야 합니다.
힌트:
  • open() 함수로 파일을 열고 read(), write()로 읽고 쓸 수 있습니다.
  • with 구문을 쓰면 파일을 자동으로 닫아줍니다.
  • try, except으로 파일 예외를 처리할 수 있습니다.
  • 파일을 한 줄씩 읽으려면 for line in file: 또는 readlines()를 사용할 수 있습니다.
  • 파일 모드는 'w'(쓰기), 'r'(읽기), 'a'(추가) 등이 있습니다.
