9주차 - 파일 입출력

  1. examples_week9.txt 파일이 있다고 가정해서 진행
  2. 텍스트 파일을 생성하고 몇 줄의 텍스트를 저장하세요.
  3. 저장한 파일을 열어 내용을 읽고 출력하세요.
  4. 파일이 없을 때 발생하는 예외를 처리하세요 (try/except).
  5. 파일의 각 줄을 읽어 리스트에 저장한 뒤 길이를 출력하세요.
  6. 파일에 덧붙이기(append) 모드로 한 줄을 추가하세요.
힌트:
  • open() 함수로 파일을 열고 read(), write()로 읽고 쓸 수 있습니다.
  • with 구문을 쓰면 파일을 자동으로 닫아줍니다.
  • try, except으로 파일 예외를 처리할 수 있습니다.
  • 파일을 한 줄씩 읽으려면 for line in file: 또는 readlines()를 사용할 수 있습니다.
  • 파일 모드는 'w'(쓰기), 'r'(읽기), 'a'(추가) 등이 있습니다.
