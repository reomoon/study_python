6주차 - 함수/모듈

  1. 1) 두 수를 입력받아 더하는 함수 add(a, b)를 작성하고, 결과를 출력하세요.
    예: add(2, 3) → 5 출력
  2. 1) 문자열을 입력받아 뒤집는 함수 reverse(s)를 작성하세요.
    예: reverse('hello') → 'olleh' 출력
  3. 1) 기본값 인자를 사용하는 함수를 작성하세요.
    2) 인자를 생략하면 기본값이 사용되도록 하세요.
    예: def greet(name='학생')
  4. 1) 함수 내부에 문서화 문자열(docstring)을 작성해서 __doc__가 채워지도록 하세요.
    예: print(func.__doc__)
  5. 1) 리스트를 입력받아 최댓값을 반환하는 함수 find_max(lst)를 작성하세요.
    예: find_max([1,5,3]) → 5 출력
  6. 1) 입력된 정수가 소수(prime)인지 판별하는 함수 is_prime(n)을 작성하세요.
    2) 소수면 True, 아니면 False를 반환하세요.
    예: is_prime(7) → True, is_prime(8) → False
  7. 1) 리스트에서 특정 값의 개수를 세는 함수 count_value(lst, value)를 작성하세요.
    예: count_value([1,2,2,3], 2) → 2 출력
힌트:
  • def로 함수를 정의합니다.
  • return으로 값을 반환.
  • import로 모듈을 불러올 수 있습니다.
  • *args, **kwargs로 가변 인자 사용.
  • 함수에 docstring을 달면 설명이 추가됩니다.
  • 최댓값은 반복문이나 max() 함수를 활용할 수 있습니다.
  • 소수 판별은 2부터 n-1까지 나누어 떨어지는지 확인하면 됩니다.
  • 리스트에서 특정 값의 개수는 count() 메서드나 반복문으로 셀 수 있습니다.
