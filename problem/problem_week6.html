6주차 - 함수/모듈

  1. 두 수를 더하는 함수 add(a,b)를 작성하고 결과를 출력하세요.
  2. 문자열을 뒤집는 함수 reverse(s)를 작성하세요.
  3. 아래 내용 참고하여 print_gugudan을 import 후 구구단 3단을 입력하세요.
    def print_gugudan(dan):
    for i in range(1, 10):
    print(f"{dan} x {i} = {dan * i}")
    그리고 다른 파일에서 from gugudan import print_gugudan으로 함수를 불러와 print_gugudan(3)을 실행해 3단을 출력하세요.
  4. 기본값 인자를 사용하는 함수를 작성하세요.
  5. 함수 내부에서 문서를 작성하여 __doc__가 채워지도록 해보세요.
힌트:
  • def로 함수를 정의합니다.
  • return으로 값을 반환.
  • import로 모듈을 불러올 수 있습니다.
  • *args, **kwargs로 가변 인자 사용.
  • 함수에 docstring을 달면 설명이 추가됩니다.
